# productivity/ — everyday helper skills

Small, standalone skills — no shared machinery or dependency between them, unlike `workflow/`'s
generator+orchestrator pair. Each one just does its own focused job.

| Skill | What it does |
|---|---|
| [`givename/`](givename/) | Suggests 5 names (shortest → longest) for anything — variables, files, git branches/commits, new projects/skills — by reading the actual naming convention nearby first. See [`givename/README.md`](givename/README.md). |
